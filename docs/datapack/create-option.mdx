# Create Option

## Description

Create Options show up when the + Button is pressed.They can be found/added in mod-id:miapi/create_options/any-path-and-file-name.json### `item` The id of the item to be created, usually "miapi:modular_item" will work.### `module`The id of the base module to be added.### `translation`The Translation key of the name### `count`How many will be created at once
