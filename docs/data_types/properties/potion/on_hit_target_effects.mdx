# On Hit Target Effects Property

## Description

  
This property triggers specific potion effects on the target entity when the attacker hits it. The effectsare defined by `PossibleEffect` instances, which include parameters like probability, duration, and amplifier.The effects can be configured to apply based on various conditions, and the tooltip will display relevantinformation about these effects.

## Data

- **effects**:  A list of `PossibleEffect` instances that define the potion effects to be applied when hitting the target entity.
