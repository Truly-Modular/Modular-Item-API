# On Damaged Effects Property

## Description

  
This property triggers specific potion effects on an entity when it is damaged. The effects and theirproperties (such as duration, amplifier, probability) are defined by `PossibleEffect` instances. The effectscan be configured to apply to either the attacker or the target entity, based on the `targetSelf` parameter.

## Data

- **effects**:  A list of `PossibleEffect` instances, each defining a potion effect to be applied under certain conditions.
