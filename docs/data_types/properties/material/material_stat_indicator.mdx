# Material Stat Indicator Property

## Description

  
The Material Stat Indicator Property is used to define and store various material-related attributes for items.It allows assigning specific stats such as durability, hardness, and mining speed, which can be represented witha strength value and an optional description. If no description is provided, a default localized description is used.

## Data

- **material_stat_indicator**:  A map of material statistics, where each entry consists of a stat name and a context object containing its strength and optional description.
