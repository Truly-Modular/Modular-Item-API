# Staggered Mining

## Description

  
This mode processes blocks in a staggered manner, giving a more gradual mining experience compared to instant mining. It handles mining tasks in a delayed fashion, which can help manage performance and resource usage during mining operations.

## Data

- **speed**:  The rate at which blocks are mined. Higher values result in faster mining speeds.
- **durabilityBreakChance**:  The probability that the item's durability will decrease during mining, ranging from 0.0 (no chance) to 1.0 (guaranteed breakage).
