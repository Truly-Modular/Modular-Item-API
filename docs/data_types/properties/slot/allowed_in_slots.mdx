# Allowed Slots Property

## Description

  
The Allowed Slots Property controls where a module can be placed by defining which slot types it is compatible with.A module can specify multiple slots by their IDs, ensuring that only certain modules can fit into designated slotswhen assembling items. This helps define logical compatibility between modules and item slots.

## Data

- **allowed_in_slots**:  A list of strings representing the slot IDs where the module can be placed.
- **`slotId`**:  (required) The ID of the slot that allows this module.
