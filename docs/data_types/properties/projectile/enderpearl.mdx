# Enderpearl Property

## Description

  
The Enderpearl Property allows modular projectiles to mimic the teleportation behavior of ender pearls. When aprojectile with this property hits an entity or block, it teleports the owner (if possible) to the point of impact,similar to the effect of throwing an ender pearl.

## Data

- **is_enderpearl**:  A boolean value that determines if the projectile behaves like an ender pearl.
