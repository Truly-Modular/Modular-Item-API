# Fire Proof Property

## Description

  
The FireProof property allows items to be marked as fireproof, making them immune to Lava. When an item is set with thisproperty, it will not be damaged or affected by fire or lava.This property is represented as a boolean value where `true` indicates that the item is fireproof, and `false` means it is not.

## Data

- **fireProof**:  A boolean value indicating whether the item is immune to fire and lava.
