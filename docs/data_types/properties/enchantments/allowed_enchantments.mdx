# Allowed Enchantments Property

## Description

  
The Allowed Enchantments Property is used to specify and manage enchantments that can or cannot be applied to certain items.It handles enchantment restrictions based on item types and also allows extensions to be detected and dynamically added.This property is crucial for controlling item enchantment compatibility within the modular item system.

## Data

- **allowed**:  a list of allowed enchantments (ResourceLocation).
- **forbidden**:  a list of forbidden enchantments (ResourceLocation).
