# Crafting Enchantments Property

## Description

 The Crafting Enchantments Property adds specific enchantments to an item when it is crafted.  
These enchantments are permanently added to the item and remain during subsequent uses or modifications.The property dynamically calculates the level of the enchantment using a resolvable operation that can reference the previous level of the enchantment.It integrates with the core enchantment system and ensures compatibility with existing enchantment handling mechanisms.

## Data

- **enchantment**:  The enchantment being applied.
- **value**:  Double Resolvable, used to calculate the level of the enchantment.
