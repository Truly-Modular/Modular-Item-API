# Emissivity Property

## Description

  
The EmissivityProperty defines the emissive light levels for an item or block. Emissive properties affect how much lightan item or block emits.These values are applied to the item or block to control its visual appearance in lighting conditions.

## Data

- **sky**:  An integer representing the light level emitted in the sky.
- **block**:  An integer representing the light level emitted from the block.
