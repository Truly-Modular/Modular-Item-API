# Item ID Property

## Description

 The ItemIdProperty allows for the modification of an item's identifier upon crafting. When this property is applied,  
it changes the `ItemStack`'s item to a different one based on a in java registered Modular Item.This is useful for dynamically altering items during the crafting process, giving you the ability to switch item typesaccording to predefined configurations.

## Data

- **item_id**:  the desired Item ID.
